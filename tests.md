# Тестирование приложения Currency Converter

**Проект:** Currency Converter  
**Автор:** Вахиуллин Артур Ильдарович

## 1. Проверяемые функции

- запуск графического интерфейса;
- выбор валют;
- ввод суммы;
- конвертация валют;
- получение курса через API;
- обработка ошибки API;
- сохранение истории в JSON;
- загрузка истории из JSON;
- очистка истории;
- проверка корректности ввода.

## 2. Позитивные тесты

| № | Действие | Входные данные | Ожидаемый результат | Результат |

| 6 | Сохранение истории | Выполнить конвертацию | Запись появляется в `history.json` | Успешно |
| 7 | Загрузка истории | Перезапустить программу | История загружается в таблицу | Успешно |
| 8 | Очистка истории | Нажать кнопку очистки | Таблица и JSON очищаются | Успешно |

## 5. Проверка JSON

Пример структуры файла `history.json`:

```json
[
  {
    "date": "2026-04-28 10:15:00",
    "amount": 10.0,
    "from_currency": "USD",
    "to_currency": "RUB",
    "result": 909.09,
    "rate": 90.909091
  }
]
```

Проверено:

- история сохраняется в JSON;
- история загружается при запуске;
- после очистки история обновляется;
- данные можно просмотреть вручную.
