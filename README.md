# Currency Converter — Конвертер валют

**Автор:** Вахиуллин Артур Ильдарович

## Описание

Currency Converter — это графическое приложение на Python с использованием библиотеки `tkinter`.  
Программа позволяет конвертировать сумму из одной валюты в другую, получать актуальные курсы через внешний API, сохранять историю операций в JSON и загружать её обратно.

## Возможности

- выбор исходной валюты;
- выбор конечной валюты;
- ввод суммы;
- проверка корректности суммы;
- конвертация валют;
- получение курса через внешний API;
- резервные курсы на случай отсутствия API-ключа или интернета;
- сохранение истории в `history.json`;
- загрузка истории из JSON;
- очистка истории;
- таблица истории конвертаций.

## Требования

- Python 3.7 или выше;
- библиотека `requests`.

Установка зависимости:

```bash
pip install requests
```

## Как получить API-ключ

1. Перейдите на сайт exchangerate-api.com.
2. Зарегистрируйтесь или войдите в аккаунт.
3. Получите бесплатный API-ключ в личном кабинете.
4. Откройте файл `main.py`.
5. Найдите строку:

```python
API_KEY = "YOUR_API_KEY"
```

6. Замените `YOUR_API_KEY` на свой ключ:

```python
API_KEY = "ваш_api_ключ"
```

После этого программа будет получать актуальные курсы валют через API.

## Как запустить

1. Скачайте проект.
2. Откройте папку проекта в терминале.
3. Установите библиотеку:

```bash
pip install requests
```

4. Запустите приложение:

```bash
python main.py
```

## Примеры использования

### Пример 1. Конвертация USD в RUB

1. Введите сумму: `10`.
2. Выберите валюту `USD`.
3. Выберите валюту `RUB`.
4. Нажмите кнопку **Конвертировать**.
5. В таблице истории появится новая запись.

### Пример 2. Конвертация EUR в RUB

1. Введите сумму: `5`.
2. Выберите `EUR`.
3. Выберите `RUB`.
4. Нажмите кнопку **Конвертировать**.

### Пример 3. Ошибка ввода

Если ввести `abc` или отрицательное число, программа покажет сообщение об ошибке.
